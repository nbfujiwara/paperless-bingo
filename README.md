# Quick Start

1. ```common/.env.local.sample```　をコピーし、Firebase情報を記入して　```common/.env.local```　として保存
1. パッケージインストール
```
$ yarn install
```
1. ローカルで確認（プレイヤー側）
  ```
$ cd app_player
$ yarn dev
```
http://localhost:3000　にアクセス
1. ローカルで確認（管理側）
```
$ cd app_admin
$ yarn dev
```
http://localhost:3001　にアクセス
